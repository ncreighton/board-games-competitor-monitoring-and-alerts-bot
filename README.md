# Board Games Competitor Monitoring and Alerts Bot

A comprehensive competitor monitoring and alerts bot designed specifically for the board games industry.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
